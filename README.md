## Getting Started

### Installation

- Clone This Front End Repo

```
git clone https://github.com/Rdhwnzaki/Frontend-FoodRecipe.git
```

- Go To Folder Repo

```
cd Frontend-FoodRecipe
```

- Install Module

```
npm install
```
